## Тема
Основы трансформации данных

## Что сделано
Выполнено преобразование данных температурных показаний IOT-устройств с помощью Apache Airflow.

## Структура проекта
- `data/IOT-temp.csv` — исходный датасет
- `dags/process_iot_temp_dag.py` — DAG Airflow
- `output/cleaned_iot_temp.csv` — очищенные данные
- `output/top_5_hottest_days.csv` — 5 самых жарких дней
- `output/top_5_coldest_days.csv` — 5 самых холодных дней

## Что выполнено по заданию
- отфильтрованы строки, где `out/in = In`
- поле `noted_date` преобразовано в формат `yyyy-MM-dd`
- температура очищена по 5-му и 95-му процентилю
- вычислены 5 самых жарких и 5 самых холодных дней за год

## Результат
DAG `process_iot_temperature_data` успешно выполнен в Airflow.
